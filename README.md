# Dependencies
* Core Tools Version: 4.0.5148
* Function Runtime Version: 4.17.3.20392

**PRE REQUESITES**
* Install node modules 
    ```
    npm install
    ```

* Update local.settings.json with actual values
* Run the application
    ```
    npm run start
    ```
